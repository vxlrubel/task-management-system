# Task Management System (Asana-Like Frontend)

A modern **frontend-only task management system** inspired by **Asana**, built to manage tasks, projects, teams, and workflows using API-based architecture.

This project focuses on clean UI/UX, scalable state management, and role-based access, while keeping the backend completely decoupled.
